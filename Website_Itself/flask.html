In this section I will help with Flask installation and using it in the Visual Studio Code.

Before we start, I want you to understand that we are going to use Flask in Python, so if you don't have one, it is better to download it.


First of all, we need to open the Command Prompt. You can do it by searching it in the Windows searchbar and typing CMD.


Then we are going to change directory. It is better to change to the folder where are you going to work later. I will create my own and call it workplace.

>> cd C:\Users\admin\Desktop\files

After creating one, we are going to write down next command to create a folder venv.

>> py -3 -m venv venv

Before working on a project, we should activate the corresponding environment.

>> venv\Scripts\activate

And within it, we can fianlly install Flask.

>> pip install Flask

You can open your folder woth intalled files in VS Code and work with no problems.

But don't forget to check interpreters.



And congratulations, you have succesfully installed it. Now we can proceed and check how to work with it in the Quickstart section.
