Firstly, in HTML we change "count" button into mathematical symbols, such as plus or minus. Now let's see the JS code.

The full JavaScript code looks like this:

let num1, num2, out;

function add() {
    num1 = parseInt(document.getElementById("number1").value);
    num2 = parseInt(document.getElementById("number2").value);
    if ((/^\d+$/.test(num1) == false) || (/^\d+$/.test(num1) == false)) {
        out = "Error"
    }
    else {
        out = num1 + num2;
    }
    document.getElementById("result").value = out;
}


function minus() {
    num1 = parseInt(document.getElementById("number1").value);
    num2 = parseInt(document.getElementById("number2").value);
    if ((/^\d+$/.test(num1) == false) || (/^\d+$/.test(num1) == false)) {
        out = "Error"
    }
    else {
        out = num1 - num2;
    }
    document.getElementById("result").value = out;
}


function division() {
    num1 = parseInt(document.getElementById("number1").value);
    num2 = parseInt(document.getElementById("number2").value);
    if ((/^\d+$/.test(num1) == false) || (/^\d+$/.test(num1) == false)){
        out = "Error"
     }
    else {
        out = num1 / num2;
    }
    document.getElementById("result").value = out;
}

function mult() {
    num1 = parseInt(document.getElementById("number1").value);
    num2 = parseInt(document.getElementById("number2").value);
    if ((/^\d+$/.test(num1) == false) || (/^\d+$/.test(num1) == false)) {
        out = "Error no integers were found"
    }
    else {
        out = num1 * num2;
    }
    document.getElementById("result").value = out;
}

It can be seen that we have created three let type objects.

Is it known, that in JavaScript there are several object types:

● var – containers for storing data.

● const – unchangable storage.

● let – changeable storage, best suits math algorythms.


In the HTML code, there were created two windows with option to fill it with numbers.

Each of these windows has names and it's values. So, the first number just called: number1.

After using the command document.getElementById("").value, we can find the given value by searching the same string name.

Thus, by writing document.getElementById("number1").value we are finding the name and then it's value.


It was also added if else function, which checks if there are intergers:

if ((/^\d+$/.test(num1) == false) || (/^\d+$/.test(num1) == false)) {
    out = "Error no integers were found"
}
else {
    out = num1 + num2;
}

Thanks to this check the user will be informed, that the inputs are incorrect.

To be more precise this command /^\d+$/.test(num1) tells if our variable contains only numbers. So, if it's false the programm gives us an Error.
